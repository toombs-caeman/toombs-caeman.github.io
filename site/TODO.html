About this site

This site is served using github pages for convenience. It is entirely static: No server side processing, no client side javascript (for now). Pages are rendered as part of the git pre-commit hook. You can view the project repo here.

If I ever have to move away from github for some reason, the site could be hosted as simply as nginx or httpd serving an up-to-date copy of the repository. Nginx logs can be captured to provide the same analytics available on github, namely unique visitors per week. I use flask as the development server, but that's just me being lazy.
